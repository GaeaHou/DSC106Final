Daily Glucose Visualization

Project Writeup

1. What have we done so far?

So far, we have built the structural framework of the webpage and completed the introductory animation section.We loaded and parsed the cleaned CSV, extracted HbA1c, grow_in_glu, and Gender fields, and computed three equally sized HbA1c groups (Low/Mid/High).We Set up SVG canvas with appropriate margins and scales,wrote a kernel density estimator and computed density curves for each HbA1c group in violin plot and Grouped data by gender (Male/Female) and calculated five-number summaries (Q1, Median, Q3, whisker bounds) in boxplot.Drew whiskers, boxes, and median lines using SVG primitives.We have also preprocessed the dataset and manually corrected issues with formatting and inconsistent values. Based on the cleaned data, We trained a model to simulate glucose trends in response to different foods. Using this model, We created a visualization that shows how glucose levels change across different individuals after eating. Additionally, we designed and implemented the user interface for selecting specific individuals and dates. The page also includes a chart that updates dynamically based on user input. This interactive design helps users explore personalized glucose responses. I also implemented tooltips and a reset function to enhance usability.
