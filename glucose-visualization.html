Daily Glucose Visualization

Project Writeup

1. What have we done so far?

So far, we have built the structural framework of the webpage and completed the introductory animation section. We have also preprocessed the dataset and manually corrected issues with formatting and inconsistent values. Based on the cleaned data, We trained a model to simulate glucose trends in response to different foods. Using this model, We created a visualization that shows how glucose levels change across different individuals after eating. Additionally, we designed and implemented the user interface for selecting specific individuals and dates. The page also includes a chart that updates dynamically based on user input. This interactive design helps users explore personalized glucose responses. I also implemented tooltips and a reset function to enhance usability.
